# DriveManager

This repository contains the Drive Manager microservice.

Refer to [API_DOCUMENTATION.md](API_DOCUMENTATION.md) for a complete description of the REST API, including endpoint definitions and a workflow diagram.

## Quick start

```bash
./mvnw spring-boot:run
```

Once running, you can use any HTTP client to call the endpoints documented in `API_DOCUMENTATION.md`.

> **Tip:** Save the attached workflow diagram in `docs/api_flow.png` and open the markdown file to view it inline.
